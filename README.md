### Dependencies
Install:
* [https://nodejs.org/en/download/](node)
* [https://www.npmjs.com/get-npm](npm)

### Setup
1) Clone this repo
2) 'cd' to `/my-app` directory
3) Run:
```
npm install
```

### Run app in dev mode
Run:
```
npm run start
```
...and visit [http://localhost:3000/](http://localhost:3000/)
### Run tests
Run:
```
npm run tests
```
