## Dependencies
Install:
* [https://nodejs.org/en/download/](node)
* [https://www.npmjs.com/get-npm](npm)

## Run App in dev mode
Run the following command:
```
npm run start;
```
...and visit [http://localhost:3000/](http://localhost:3000/)
## Run Tests
Run the following command:
```
npm run tests;
```
